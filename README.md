# Sefonx – Portfolio Website

A modern, responsive portfolio website built with pure HTML, CSS, and JavaScript.

## Features

- 🎨 Dark theme with red accent color
- ✨ Smooth scroll animations & typewriter effect
- 🖼️ Real tech-stack icons from devicons CDN
- 📱 Fully responsive (mobile, tablet, desktop)
- ⚡ No frameworks – pure HTML/CSS/JS

## Sections

- **Hero** – Profile photo, intro, social links
- **Stats** – Key numbers at a glance
- **About** – Bio and personal details
- **Skills** – Frontend & Backend tabs with real icons
- **Projects** – Featured work cards
- **Contact** – Contact form and info

## Setup

1. Clone the repo
2. Put your CV as `assets/cv.pdf`
3. Open `index.html` in a browser — that's it!

## Deploy to GitHub Pages

1. Push to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://USERNAME.github.io/REPO`
